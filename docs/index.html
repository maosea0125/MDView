功能特性

专注阅读体验，覆盖主流 Markdown 扩展语法

📝

完整 Markdown 支持

GFM 表格、任务列表、脚注、Emoji、Front Matter、锚点导航

📐

数学公式

KaTeX 渲染引擎，支持行内和块级 LaTeX 公式

📊

Mermaid 图表

流程图、时序图、甘特图，支持弹窗放大至 1500% 查看细节

📑

多标签页

同时打开多个文件，拖拽排序，右键快捷操作

🌗

主题切换

明亮 / 暗黑主题一键切换，跟随 GitHub 渲染风格

📂

文件关联

系统右键菜单直接打开，拖放文件，最近文件快速访问

🔍

缩放控制

Cmd/Ctrl +/- 快捷键调整字号，自由缩放预览

📋

目录导航

自动从标题提取目录，侧边栏一键跳转对应章节

⚡

原生性能

Tauri v2 + Rust 后端，安装包 < 10MB，启动秒开
